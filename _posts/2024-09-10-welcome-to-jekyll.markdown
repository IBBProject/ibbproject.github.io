# Installing an IBB

## System Requirements

All IBB Systems currently require the use of x86/64 based linux systems.

## Minimum Requirements

* 1 Processor Cores
* 4GB RAM
* 30+ GB Storage

## Recommended Requirements

* 4 Processor cores/threads
* 8GB RAM
* 50+ GB Storage
